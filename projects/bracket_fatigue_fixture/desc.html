Bracket Fatigue Fixture

This is an 800lbf cyclic fatigue fixture designed for testing Mytra's Cell Brackets. The pneumatic system directly loads both sides of the custom bracket with ~800lbf for FEA and production verification/validation testing.

The fixture performs 50k cycles per test with an expected lifetime of 2-3 years and 1M total cycles. Built with an 8020 aluminum frame for cost efficiency, this design enabled the team to construct two additional identical fixtures, tripling our testing throughput.

The control system mirrors the CT loading fixture for uniformity, utilizing the same pressure transducers and SMT-32 dev boards, with only the valve type modified for pneumatic system compliance. This standardization simplified maintenance and operation across multiple test stations.
